# yakudatsu!
# 役立つ!
Add in-line help popover text to your HTML content - Fast and Easy
## Example Usage:
### Add the CDN just above the closing `</body>` tag of your web page
`<script src="https://outendesigns.github.io/yakudatsu/yakudatsu.js"></script>`

## Special Classes Dictionary
### helper-width-[x]
Sets the width of the popup window. Default=300px<br/>
Usage example: helper-width-200<br/>
This would set the width to 200px<br/>
### helper-bgcolor-[x]
Sets the background color of the popup window. Default=#666<br/>
Usage example 1: helper-bgcolor-#123456<br/>
Usage example 2: helper-bgcolor-red<br/>
### helper-fontcolor-[x]
Sets the font color of the text within the popup window. Default=#fff<br/>
Usage example 1: helper-fontcolor-#123456<br/>
Usage example 2: helper-fontcolor-blue<br/>
### helper-fontsize-[x]
Sets the font size of the text within the popup window. Default=12px<br/>
Usage example: helper-fontsize-14<br/>
This would set the font size tto 14px<br/>
